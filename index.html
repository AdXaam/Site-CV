Mon rôle englobait un large éventail de responsabilités, de la prise de commandes à la préparation des repas, en passant par le service aux clients. Cette expérience m'a permis d'acquérir une compréhension approfondie des opérations en restauration, tout en développant ma résilience, ma réactivité et ma capacité à travailler harmonieusement au sein d'une équipe.

Langages Informatiques

Frameworks

HTML (HyperText Markup Language)
HTML est le langage de balisage utilisé pour créer et structurer le contenu des pages web. Il permet de définir la structure logique d'un document en utilisant des balises pour représenter des éléments tels que titres, paragraphes, liens, images, formulaires, etc. HTML est essentiel pour la construction de pages web et constitue la base de tout développement web.
CSS (Cascading Style Sheets)
CSS est un langage de feuille de style qui complète HTML en permettant la conception visuelle des pages web. Il définit la présentation, la mise en page et le style des éléments HTML. CSS permet de gérer la couleur, la police, la disposition et d'autres aspects visuels d'une page, offrant ainsi un contrôle esthétique et une expérience utilisateur optimisée.
Javascript
JavaScript est un langage de programmation côté client utilisé pour rendre les pages web interactives. Il permet d'ajouter des fonctionnalités dynamiques, de manipuler le contenu de la page en temps réel, de répondre aux événements utilisateur et d'interagir avec le navigateur. JavaScript est essentiel pour la création d'applications web modernes et réactives.
SQL (Structured Query Language)
SQL est un langage de requête utilisé pour gérer et interagir avec des bases de données relationnelles. Il permet de créer, lire, mettre à jour et supprimer des données dans les bases de données. SQL est fondamental pour la gestion des données dans les applications web, et il offre des fonctionnalités avancées telles que la jointure de tables, la sélection de données spécifiques, et bien plus encore.
PHP
PHP est un langage de script côté serveur conçu pour le développement web. Il est principalement utilisé pour créer des pages web dynamiques en générant du contenu à partir du serveur. PHP peut se connecter à des bases de données, gérer des formulaires, traiter des cookies, et effectuer diverses opérations côté serveur. Il est largement utilisé dans le développement web côté serveur.
Typescript
TypeScript est une extension de JavaScript qui ajoute des fonctionnalités de typage statique au langage. Il permet de détecter les erreurs potentielles plus tôt dans le processus de développement en spécifiant explicitement les types de données. TypeScript est souvent utilisé dans des projets de grande envergure pour améliorer la maintenabilité du code, la productivité de l'équipe de développement et la qualité du logiciel final.
Angular
Angular est un framework de développement web côté client basé sur TypeScript. Il est développé et maintenu par Google. Angular offre une structure robuste pour la création d'applications web dynamiques et interactives. Grâce à son architecture modulaire, Angular facilite la création de composants réutilisables, la gestion de l'état de l'application et l'intégration aisée de fonctionnalités telles que le routage et la gestion des formulaires. Il encourage également le développement en utilisant le paradigme de programmation orientée composant.
Symfony
Symfony est un framework PHP côté serveur qui suit le modèle de conception MVC (Modèle-Vue-Contrôleur). Il offre une structure organisée pour le développement d'applications web robustes et évolutives. Symfony facilite la gestion des routes, la création de contrôleurs, et l'interaction avec la base de données grâce à un ORM (Object-Relational Mapping) appelé Doctrine. Symfony met l'accent sur la réutilisabilité du code, la modularité et la facilité de maintenance, ce qui en fait un choix populaire pour des projets web de grande envergure.
